# Rock Paper Scissors

This project was created as part of the 'JavaScript Basics' section of [The Odin Project](https://www.theodinproject.com/)'s 'Foundations' course.

Its main purpose was to practice fundamental concepts, like handling user interaction and implementing game logic.

The initial version didn't have a user interface, and the game was entirely played in the console. Later in the curriculum, we had to revisit the project to add a fully interactive UI, and make the game playable with buttons.
